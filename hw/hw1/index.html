What is Linux?


Linux is an operating system! An operating system is software that manages the hardware in which your desktop or laptop run. Other operating systems that may come to mind are Windows 10 or Mac 0S X.

Why Linux?

Linux is free! You can install Linux on as many computers as you like without having to pay for software or server licensing! Most importantly Linux comes in various distributions! Which in essence means there are different versions of Linux to suit various user needs and preferences.

Components of Linux?

While there are various different types and version of Linux. There are some very common features among all distributions of Linux. Some features include:

  • The Bootloader
  • The kernel
  • Daemons
  • Graphical Server
  • Desktop Environment
  • Applications

Linux Distributions?

While there are hundreds of Linux distributions out there, some are more popular than others. To list three popular Linux distribution (without any particular order), include: Ubuntu, Arch Linux and Debian.
