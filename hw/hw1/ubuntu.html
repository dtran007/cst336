Ubuntu

Ubuntu is an ancient African word meaning "humanity to others". In encompassing the spirit of Ubuntu to the world of computers and software, Ubuntu has a scheduled release every six months. Noteable about Ubuntu is that it contains many flavours and specialized derivatives.
Some recognized flavours include:

  • Edbuntu- Ubuntu for Education
  • Kubuntu- Ubuntu with KDE desktop environment
  • UbuntuKylin- Ubuntu localised for China
  • Xubuntu- Ubuntu with the XFCE desktop environment
  • Ubuntu Studio- Designed for multimedia editing and creation
  • Mythbuntu- Designed for creating a home theatre PC with MythTV
