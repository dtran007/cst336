Arch Linux

Arch Linux is a GNU/Linux distribution that aims to provide the latest stable versions of most software by following a rolling-release model. Arch Linux has a philosophy on striving to be minimalistic and pragmatic as this distribution is very user-centric for customising the operating system to user need. Arch Linux maintains extensive documentation through their wiki page.

For more information about Arch Linux visit ArchWiki
